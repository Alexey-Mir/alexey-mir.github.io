1. Концептуальное проектирование

  • Категория: Работа
  • Описание: Разработка основных технических решений по энергетике для концептуальных проектов
  • Примечания: Более 6 лет опыта выполнения концептуальных проектов по электроснабжению

Навыки и обязанности

  • Разработка основных технических решений по энергетике для концептуальных проектов (ИПРР, ИМ) на новых и действующих нефтяных и газовых месторождениях, оценка капитальных вложений и операционных затрат по вариантам электроснабжения;

  • Проработка вариантов инфраструктурных решений для обеспечения необходимого полезного использования ПНГ;

  • Защита принятых решений перед службами заказчика, экспертными и контролирующими организациями;

  • Экспертиза сторонних концептуальных проектов;

  • Разработка и актуализация цифровых информационных моделей;

  • Ведение расчетов энергетических систем с применением как специализированного ПО, так и собственных программных разработок;

  • Автоматизация рабочих процессов с использованием Python и VBA;

  • Работа с презентационными материалами, существующими и перспективными схемами электроснабжения, экономическими моделями.
